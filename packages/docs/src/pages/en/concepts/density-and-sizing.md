# Density and sizing

## Size

Reduces overall dimensions, padding, and font size.

- x-small
- small
- default
- large
- x-large

## Density

Reduces vertical padding and sometimes font size. Square/round components like icons will also reduce horizontal padding.

- default
- comfortable
- compact
